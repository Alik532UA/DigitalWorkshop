# Звіт про аудит термінології (UA)

Цей звіт містить аналіз розбіжностей у термінології проекту DigitalWorkshop та пропозиції щодо їх уніфікації для покращення UX та консистентності бренду.

## 1. Виявлені розбіжності

| Об'єкт | Виявлені назви | Контекст |
| :--- | :--- | :--- |
| **Соціальний блок** | Акція, Соціальна ініціатива, спеціальна пропозиція | Навігація, Заголовки сторінок, Кнопки в Hero |
| **Apps** | Застосунки, додатки | Навігація, Текст Hero, Кнопки |
| **Процес замовлення** | Замовити сайт, Залишити заявку | Кнопки CTA, Футер |

---

## 2. Варіанти уніфікації

### Варіант А: "Професійно-соціальний" (Рекомендовано)
Цей варіант підкреслює експертність та благородну мету.
- **Apps:** "Застосунки" (скрізь). Це звучить сучасно та професійно.
- **Social:** "Соціальна програма". У меню — "Програма", заголовок — "Соціальна програма", кнопка в Hero — "соціальна програма".
- **Чому:** Усуває комерційний відтінок слова "Акція".

### Варіант Б: "Дружній та зрозумілий"
Максимально спрощена мова для широкої аудиторії.
- **Apps:** "Додатки" (скрізь). Найпопулярніший термін серед користувачів.
- **Social:** "Допомога школам". У меню — "Допомога", заголовок — "Допомога творчим школам", кнопка в Hero — "безкоштовна допомога".
- **Чому:** Одразу пояснює суть пропозиції без зайвого пафосу.

### Варіант В: "Креативно-технічний"
Акцент на унікальності та продукті.
- **Apps:** "Web-застосунки". Підкреслює технологічний стек (Svelte).
- **Social:** "Творча ініціатива". У меню — "Ініціатива", заголовок — "Творча ініціатива для шкіл", кнопка в Hero — "творча ініціатива".
- **Чому:** Створює спільне поле між творчістю (школи) та розробкою.

---

## 3. Рекомендації щодо впровадження
1. Обрати **Варіант А** як найбільш збалансований.
2. Оновити файл `uk.ts` відповідно до обраного варіанту.
3. Перевірити, щоб англійська локалізація (`en.ts`) також відповідала обраному тону (наприклад, "Social Program" замість "Promo").
